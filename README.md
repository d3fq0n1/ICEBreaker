# ICEBreaker

**Civic Mirror. Zero Apologies. Full Send.**

Developed by **Blake Clark**  
Mauston, Wisconsin  
2025  

> “This tool exists because I got tired of watching shadows move unchecked.”

ICEBreaker is a free, encrypted Android app for **witnessing** — not violence, not evasion, but **evidence**.  
It logs, maps, and stores sightings of federal enforcement activity for public awareness and historical memory.

---

## ⚠️ Legal Notes

- No data is collected.
- No personal information is stored.
- All logs are local, encrypted, and temporary by design.

If you think this app is dangerous, **ask yourself why public accountability scares you**.
